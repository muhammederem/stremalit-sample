# Streamlit Sample Project for ML

Basic ML Training and Testing UI Desing with Streamlit

### Install Requirements

```bash
pip install -r requirements.txt
```

### To run training

```bash
streamlit run train.py
```

### To run single test

```bash
streamlit run test.py
```
